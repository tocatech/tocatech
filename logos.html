Logo variations for Tocatech smart property protection. All logos are inline SVG — scalable, no external dependencies.

How to Use

  • Right-click any logo and choose Inspect, then copy the <svg> element
  • Or view page source and copy the SVG block you need
  • All SVGs use viewBox for infinite scaling — resize via CSS width/height
  • Brand blue: #0B74D8  |  Dark: #111827
