🔊 Sound Sensor
Noise Thresholds
Set separate alert thresholds for daytime and night time. 0–100 scale (100 = maximum noise 100db).
Time Periods
Define when day and night begins. Daytime threshold applies between daytime start and night start.
e.g. Day starts 07:00, Night starts 23:00 → daytime threshold applies 07:00–23:00, night threshold 23:00–07:00.

🏠 Rental Mode ▼
When rental mode is active, motion alerts are automatically disabled during guest stays. Smoke and sound sensors remain active.
📆 Manual Guest Dates
Add booking periods as a fallback. When a calendar feed is connected and working, the calendar takes priority. If your calendar feed is unreachable or fails to load, the system automatically falls back to these manual dates.
Click a start date then an end date to add a booking. Click a booked date to remove it.
📅 Rental Platform Calendars
Add iCal links from Airbnb, Booking.com, and other platforms. Tocatech syncs these every 30 minutes and disables motion alerts when any calendar shows a booking. If no calendar is added, or if a feed is temporarily unreachable, the system uses your manual dates above as a fallback.

📡 ESP-NOW Peripherals ▼

Peripheral sensors connected to this device via ESP-NOW. Premium subscription required for peripherals to be active.

Pair a New Peripheral

  1. Tap the BOOT button on the main device 7 times quickly (within 3 seconds) — LED blinks rapidly
  2. Tap the BOOT button on the peripheral device 7 times quickly (within 3 seconds)
  3. Both devices are now in pairing mode for 30 seconds
  4. Once paired, the peripheral appears in this list below
  5. Assign a location label (e.g. "Bedroom 1", "Front Door") and save
